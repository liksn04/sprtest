# ❄️ 삿포로 겨울 여행 가이드

2027 삿포로 눈축제 & 조잔케이 온천 3박 4일 — 나만의 모바일 여행 가이드 앱.

- **여행**: 2027.2.4(목) ~ 2.7(일) · 부산(김해) ↔ 신치토세 · 2인 · 총예산 300만원
- **코스**: 삿포로 눈축제(오도리·스스키노) → 조잔케이 온천 료칸 1박 → 오타루 운하

## 기능

| 탭 | 내용 |
|---|---|
| 홈 | 출발 카운트다운, 여행 요약, 2월 삿포로 날씨·복장 |
| 일정 | Day 1–4 타임라인, 구글맵 연결, 항목별 완료 체크 |
| 예산 | 카테고리별 예산 대비 지출 입력(자동 저장), 엔화 계산기 |
| 체크 | 예약·출국 준비·짐싸기·온천 체크리스트(자동 저장) |
| 가이드 | 눈축제 3개 회장, 온천 에티켓, 먹킷리스트, 교통, 여행 일본어 |

체크·지출 데이터는 브라우저 `localStorage`에 저장됩니다.
아이폰 사파리에서 **공유 → 홈 화면에 추가**하면 풀스크린 앱처럼 쓸 수 있어요.

## 기술 스택

Vite · React 19 · TypeScript · Tailwind CSS v4 · Motion(Framer Motion) · lucide-react

## 개발

```bash
npm install
npm run dev     # 개발 서버
npm run build   # 프로덕션 빌드 (dist/)
```
